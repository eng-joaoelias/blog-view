# Blog View

Este é um projeto simples de visualização de blog, desenvolvido como parte de um desafio do Frontend Mentor. Foi construído principalmente com HTML e CSS, e inclui um pouco de JavaScript como um bônus.

## Descrição

Este projeto consiste em um layout de visualização de blog responsivo, onde os usuários podem ler artigos e ver informações sobre a data de publicação e o autor. Ele foi desenvolvido como uma prática para melhorar as habilidades em HTML, CSS e JavaScript.

## Recursos

- Layout responsivo construído com HTML e CSS.
- Utilização de JavaScript para exibir dinamicamente a data de publicação.
- Projeto desenvolvido como um desafio de aprendizagem do Frontend Mentor.

## Como Usar

1. Clone este repositório.
2. Abra o arquivo `index.html` em seu navegador web.
3. Explore o layout do blog e a interação com a data de publicação.
